商品を簡単に検索しブロック作成。

== Description ==

Amazon Product Advertising APIを利用して、投稿画面のブロックから商品を検索し、埋め込みます。
※このプラグインは外部サービスのAmazon Product Advertising APIを使用している為、Amazon Product Advertising APIの仕様変更により、アップデートが必要になることがあります。

== Installation ==

1. `/wp-content/plugins/fragment-amzn` ディレクトリへファイルをアップロード、またはプラグインの新規追加画面から検索してインストールしてください。
2. プラグイン画面で有効化した後、アクセスキー・シークレットキー、またアフィリエイトIDの設定を行ってください。
